WagonsRecognition
=================

Project for railway wagons number recognition. Uses OpenCV library (C++ interface), IDE - Visual Studio 2010.
Developed for Video Internet Technologies Ltd. (www.vit.ua)

Eligible for ex-USSR (CIS countries) standard.
